# 人生資金シミュレーター

将来必要な資産と見込み資産をワースト・メイン・アップサイドの3シナリオで比較し、
モンテカルロシミュレーションで確率的な成功率も算出する家族共有型ライフプランツールです。

> **免責：** 本アプリは一般的なシミュレーションツールです。特定の金融商品・投資・税務戦略を推奨するものではありません。

## セットアップ

```bash
# 依存パッケージのインストール
npm install

# 開発サーバー起動
npm run dev
# → http://localhost:3000

# テスト実行
npm test

# ビルド
npm run build
```

### Supabase連携（任意）

`.env.local` にSupabaseのURLとAnon Keyを設定すると、クラウド保存・共有リンク機能が有効になります。
未設定の場合はlocalStorageのみで動作します。

```env
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
```

Supabaseのスキーマは `supabase/migrations/0001_initial_schema.sql` を参照してください。

## 計算ロジック概要

### 3期間モデル

キャッシュフローを以下の3期間に分けて計算します：

| 期間 | 定義 |
|------|------|
| **就業期** `employment` | 現在〜退職年齢 |
| **空白期** `gap` | 退職〜65歳（退職が65歳未満の場合のみ） |
| **年金期** `pension` | 65歳〜想定寿命 |

### シナリオ前提値

| | ワースト | メイン | アップサイド |
|---|---|---|---|
| 利回り（標準） | 0% | 4% | 7% |
| 支出インフレ | 2.5%/年 | 1.5%/年 | 1.0%/年 |
| ゆとり費 | ゼロ（最低限のみ） | 全額 | 全額 |

### モンテカルロ

1,000回の確率的シミュレーションを実行し、P10/P50/P90の資産分布と
「想定寿命まで資産が持続する確率」を算出します。

## ディレクトリ構成

```
src/
├── app/              # Next.js App Router ページ
├── components/       # UIコンポーネント
│   ├── ui/           # 汎用プリミティブ
│   ├── layout/       # AppShell, BottomNav
│   ├── input/        # 入力フォーム群
│   ├── results/      # 結果表示コンポーネント
│   └── summary/      # サマリー・共有
├── lib/
│   ├── simulation/   # 計算ロジック（純関数）
│   ├── supabase/     # DB クライアント・クエリ
│   ├── storage/      # localStorage ラッパー
│   └── utils/        # フォーマット・定数
├── context/          # SimulationContext（グローバル状態）
└── __tests__/        # Vitest ユニットテスト
```
